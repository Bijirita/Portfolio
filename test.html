About Me

I’m a trained Microbiologist switching career paths into Web Development! I had a fulfilling career in research, public health, and pharmaceutical settings, and noticed similar obstructions across them all. Technology bottlenecks and workflow inefficiencies would add challenges to scaling projects, increase process duplications, and be a sink for employee hours. To address this, I became determined to learn to code to create better tools for my field. I began using online resources, such as coursera and codecademy, after work to learn code. After about 6 months, I decided to immerse myself in a collaborative learning environment and joined theClubhou.se’s code bootcamp. This program was an intensive 3 month program where I learned new skills as a Full Stack Developer. In doing so, I gained a deep appreciation for the open source community, and developed my passion for user experience, data visualization, and web development!
